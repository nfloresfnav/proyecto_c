# Proyecto en Codigo C

## Descripción 
Este proyecto se realizo para aprender la diferencia entre un lenguaje compilado y un lenguaje interpretado.


## Prerequisitos
* Instalar compilador de c - apt install build-essential
* Verificar la instalación de build essential - gcc --version

# Pasos a seguir
Una vez copiado el repositorio hay que compilarlo.
gcc -conversor.c -o conversor.
Con el comando -o aparte de compilarlo lo ejecutamos directamente.
Una vez generado el programa en linux utilizamos el comando ./conversor para poderlo ejecutar.
